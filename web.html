Syllabus

Course Description

This course provides a comprehensive introduction to web technologies covering both client-side and server-side development. Students will learn to design and develop modern, responsive websites using current standards and best practices.

Course Objectives

Fundamentals

  • Understand web architecture
  • Learn core web technologies
  • Master HTML5 & CSS3

Development

  • Build interactive websites
  • Implement responsive design
  • Develop client-side scripts

Advanced Topics

  • Server-side programming
  • Database integration
  • Web security principles

Frontend Technologies

Lab Manual (15 Experiments)

Experiment 1: Basic HTML Page Structure

Learning Objectives

  • Understand basic HTML document structure
  • Practice using essential HTML tags
  • Create a valid HTML5 document

Procedure

  1. Create a new HTML file named index.html
  2. Set up the basic HTML5 document structure with doctype, html, head, and body tags
  3. Add a heading (h1) with your name
  4. Create a paragraph introducing yourself
  5. Add an image tag with your photo or a placeholder
  6. Create a simple navigation with 3 links
  7. Validate your code using the W3C Validator

Sample Code

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is my first HTML page...</p>
</body>
</html>

Expected Output

A properly structured HTML document that displays your name, introduction paragraph, and image in a web browser.

Experiment 2: CSS Styling and Layout

Learning Objectives

  • Apply CSS styles to HTML elements
  • Understand the CSS box model
  • Create a basic page layout

Procedure

  1. Create an external CSS file named styles.css
  2. Link the CSS file to your HTML document
  3. Style the body with a font family and background color
  4. Create a header section with centered text and padding
  5. Style navigation links with hover effects
  6. Implement a two-column layout using float or flexbox
  7. Add responsive design with media queries

Key Concepts to Demonstrate

  • Class vs ID selectors
  • Margin vs Padding
  • Position property
  • Flexbox basics
